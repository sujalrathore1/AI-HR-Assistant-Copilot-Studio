# AI HR Assistant using Microsoft Copilot Studio

## Project Overview

The AI HR Assistant is an enterprise-style chatbot developed using Microsoft Copilot Studio and Power Automate. The chatbot helps employees by answering HR-related queries such as leave policy, work-from-home guidelines, salary information, office timings, employee benefits, and interview processes.

The project also includes workflow automation where employees can submit leave requests directly through the chatbot. The chatbot collects employee details and triggers a Power Automate workflow that automatically sends an email notification to HR.

# Leave Request Workflow

## Workflow Process

1. Employee requests leave through chatbot
2. Chatbot asks for:
   - Employee Name
   - Leave Days
   - Leave Reason
3. Chatbot triggers Power Automate workflow
4. Outlook sends automated email notification
5. HR receives leave request request automatically
